## 素数环问题

输入一个整数n，输出所有由1~n排成的环，满足任意两个相邻数之和都是素数。  
样例输入：  
6   
样例输出：  
1 4 3 2 5 6  
1 6 5 2 3 4  
（注：输出都以1开头，其他循环的不用输出，如4 3 2 5 6 1）

思路：

```
输入n
状态选择：以环的前i个数为状态
void solve(int a[], int i) {
    if(i == n) {
        判断a是否符合素数环条件，符合就输出
    }
    for(a[i] = 1, i <= n; a[i]++) {
        solve(a, i + 1)
    }
}
调用solve(0)
```

solve和search是竞赛中最常出现的函数名称，意为用暴力方法解决问题。
